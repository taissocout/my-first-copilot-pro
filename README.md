# My First Copilot Pro

Projeto Node.js + TypeScript que simula um copiloto de desenvolvimento.

## Modos disponíveis
- **ask** → tirar dúvidas sem alterar código
- **edit** → editar trechos de código existentes
- **plan** → planejar mudanças antes de implementar
- **agent** → executar tarefas complexas autonomamente
- **study** → aprendizado e revisão de conceitos

## Uso
```bash
npm start -- ask
npm start -- edit
npm start -- plan
npm start -- agent
npm start -- study
