# MeuCopiloto

Este repositório contém **prompts personalizados** para MeuCopiloto, um assistente de desenvolvimento baseado em IA, inspirado na metodologia DIO.  

Ele ajuda a:

* **Ask**: tirar dúvidas e explicar código.  
* **Edit**: refatorar, corrigir ou melhorar trechos existentes.  
* **Plan**: criar planos de implementação revisáveis.  
* **Agent**: gerar código completo em múltiplos arquivos com boas práticas.  
* **Study**: ensinar conceitos, trade-offs e práticas de Node.js/TypeScript.
